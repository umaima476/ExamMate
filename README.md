# ExamMate
An AI Powered Examination Asssitant

# Stack
FastAPI
LLM

# Features Roadmap
# v1.0
Basic LLM integrated system that can read the content of an uploaded file and generate quiz from that based on 10 mcqs.
key functionalities
1. User can upload the ppt or pdf file.
2. Quiz will be generated by the LLM.

# v2.0
# v4.0
Regeneration Loop: ExamMate allows for Multi-Pass Generation. Teachers can lock in high-quality questions and request the AI to 'Regenerate Remaining' to add more questions.
Dynamic Template Injection Engine: Developed a Jinja2-based document generator that maps curated AI question into institutional templates, guaranteeing standardized formatting across different academic departments.
